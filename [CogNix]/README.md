# 🧠 CogNix

Uma aplicação web focada em transformar o estudo em uma experiência gamificada e personalizada. O CogNix combina aprendizado adaptativo com mecânicas de jogos para aumentar o engajamento, a consistência e o desempenho dos estudantes.

## 📌 Sobre o projeto

O CogNix é uma plataforma de estudo inteligente que entende o aluno e adapta o conteúdo de acordo com suas necessidades, transformando sua jornada de aprendizado em um sistema dinâmico e motivador.

Diferente de plataformas tradicionais, o CogNix não entrega apenas exercícios ele cria uma experiência personalizada, onde cada aluno segue sua própria trilha de evolução.

## 🎮 Gamificação

O estudo vira um jogo:

* ⭐ Sistema de XP (experiência)
* 📈 Níveis (Level 1 → 100)
* 📊 Barra de progresso
* 🎯 Missões diárias
* 🏆 Conquistas (badges)
* 🔥 Streak (sequência de dias estudando)
* 🥇 Ranking (opcional e sem exposição negativa)

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

## 🎨 Layout e Design

O design foi pensado para unir produtividade e engajamento:

* Clareza de informação
* Hierarquia visual forte
* Experiência moderna e limpa
* Engajamento através de elementos visuais

## 📱 Responsividade

O projeto conta com responsividade feita **100% com CSS puro**, adaptando-se para:

* 💻 Desktops
* 📱 Dispositivos móveis

Utilizando técnicas como:

* Flexbox
* Grid Layout
* Media Queries
