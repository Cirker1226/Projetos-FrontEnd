# 🎮 GamerZone

Uma aplicação web desenvolvida com foco no público gamer, apresentando um layout moderno, organizado e totalmente responsivo. O projeto foi construído utilizando apenas HTML e CSS, com o objetivo de praticar conceitos de estruturação, estilização e responsividade.

## 📌 Sobre o projeto

O **GamerZone** é uma página web que simula uma plataforma de conteúdos gamer, com destaque para jogos, seções organizadas e um design atrativo inspirado em interfaces modernas.

O projeto foi desenvolvido com atenção especial à experiência do usuário, utilizando um tema escuro, tipografia simples e uma disposição visual equilibrada.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3

## 🎨 Layout e Design

O layout foi pensado para transmitir uma identidade gamer, utilizando:

* Tema escuro (dark mode)
* Tipografia estilo *monospace*
* Organização em seções bem definidas
* Destaques visuais para jogos e conteúdos

## 📱 Responsividade

O projeto conta com responsividade feita **100% com CSS puro**, adaptando-se para:

* 💻 Desktops
* 📱 Dispositivos móveis

Utilizando técnicas como:

* Flexbox
* Grid Layout
* Media Queries
